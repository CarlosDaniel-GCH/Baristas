# Baristas - Cafetería

Página web de una cafetería ficticia llamada "Baristas" showcasing productos de café.

## Características

- Diseño responsivo
- Catálogo de productos de café
- Sistema de carrito de compras (UI)
- Iconos de Font Awesome
- Estilos CSS personalizados

## Estructura del Proyecto

```
Proyecto Cafeteria/
├── Proyecto.html    # Archivo principal HTML
├── estilos.css      # Estilos CSS
└── img/             # Imágenes de productos
    ├── cafe.avif
    ├── cafe-capuchino.jpeg
    ├── cafe-expreso.jpeg
    ├── cafe-irish.webp
    └── cafe-moca.webp
```

## Tecnologías

- HTML5
- CSS3
- Font Awesome 6.4.0

## Ver el proyecto

Abre `Proyecto.html` en tu navegador web.
